cg-sokoban
==========

Final work of Computer Graphics (EEL882) @ UFRJ.


System dependencies
====================

- cmake 2.8.12+ (build only)
- g++ (build only)
- SDL2
- OpenGL 2.2+
- libpng (usually already installed)
